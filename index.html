AppStore vs GooglePlay

Project Overview

When it comes to mobile ecosystems, there are two giants locked in a battle, not only for revenue, but also for the hearts and minds of developers and consumers alike. The purpose of our project is to extract data sets of Google’sPlay Store and Apple’s App Store for year 2017 from Kaggle data source , transform by normalizing both the datasets and load the datasets to a database. Ultimately, the transformed data set will allow forbetter understanding of the relative variances between vendors.
